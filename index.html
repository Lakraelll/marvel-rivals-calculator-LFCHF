How to Fill Threshold Brackets (From Game Mission Screen):
  • Open your Hero's Missions panel and check their Permanent Missions row requirements.
  • Stat Requirement: Input the absolute goal of your main performance task (e.g., if game reads "Deal 3.100 Damage", enter 3100).
  • KOs / Assists / Finals: Input the strict clear boundary of your combat task (e.g., if game reads "Land 5 Final Hits", enter 5).
  • Supports (KOs/Assists Rule): If configuring a Support character task tracking combined items (e.g., "Achieve a total of 26 KOs and Assists"), you MUST enter the halved requirement quantity into the input box instead (e.g., enter 13 for a 26 requirement).
  • Tip: Ignore the "Hero usage reaches 15 minutes" mission as the script processes session time values dynamically.
